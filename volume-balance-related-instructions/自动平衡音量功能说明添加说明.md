## 技术实现细节

### 1. 多语言支持
- **base/element/string.json**: 基础资源文件，作为默认值
- **zh_CN/element/string.json**: 中文资源文件
- **en_US/element/string.json**: 英文资源文件

### 2. 资源引用方式
在ArkTS代码中通过`$r('app.string.introduction_5')`引用字符串资源，系统会根据当前语言环境自动选择合适的字符串。

### 3. 说明界面布局
说明界面使用`ForEach`循环渲染说明数组：
```typescript
ForEach(this.introductions, (item: string) => {
  Text($r(item))
    .fontSize(13)
    .opacity(0.6)
    .lineHeight(20)
}, (item: string) => JSON.stringify(item))
```

# 自动平衡音量滑动条样式统一修改说明

## 修改目标
将自动平衡音量功能的滑动条样式修改为与音频流音量的滑动条一致，提供统一的用户体验。

## 修改前 vs 修改后对比

### 修改前的自动平衡音量滑动条
- **样式**: `SliderStyle.OutSet`（外凸样式）
- **颜色**: 
  - `.blockColor('#007DFF')`（滑块颜色）
  - `.trackColor('#E5E6EB')`（轨道颜色）
  - `.selectedColor('#007DFF')`（已选择部分颜色）
- **布局**: 简单的Slider组件
- **百分比显示**: 使用`fontWeight(FontWeight.Medium)`加粗

### 修改后的自动平衡音量滑动条（与音频流音量一致）
- **样式**: `SliderStyle.InSet`（内凹样式）
- **颜色**:
  - `.trackColor('#F1F3F5')`（轨道颜色）
  - `.selectedColor('#0A59F7')`（已选择部分颜色）
  - 无`.blockColor()`设置（使用默认滑块样式）
- **尺寸**: 
  - `.trackThickness(20)`（轨道厚度20px）
  - 包含在`Stack({ alignContent: Alignment.Top })`容器中
- **交互功能**:
  - `.focusable(true)`（可获取焦点）
  - `.defaultFocus(true)`（默认焦点）
  - `.focusOnTouch(true)`（触摸时获取焦点）
- **百分比显示**: 使用`.fontColor(Color.Black).opacity(0.6)`（与音频流音量一致）

## 具体修改内容

### 1. 滑动条样式修改
```typescript
// 修改前
Slider({
  value: this.compressionRatio,
  min: CommonConstants.MIN_COMPRESSION_RATIO,
  max: CommonConstants.MAX_COMPRESSION_RATIO,
  step: CommonConstants.COMPRESSION_STEP,
  style: SliderStyle.OutSet
})
.blockColor('#007DFF')
.trackColor('#E5E6EB')
.selectedColor('#007DFF')
.onChange((value: number) => {
  this.compressionRatio = value;
})
.width('100%')

// 修改后
Stack({ alignContent: Alignment.Top }) {
  Slider({
    value: this.compressionRatio,
    min: CommonConstants.MIN_COMPRESSION_RATIO,
    max: CommonConstants.MAX_COMPRESSION_RATIO,
    step: CommonConstants.COMPRESSION_STEP,
    style: SliderStyle.InSet,
    reverse: false
  })
  .focusable(true)
  .defaultFocus(true)
  .focusOnTouch(true)
  .trackColor('#F1F3F5')
  .trackThickness(20)
  .selectedColor('#0A59F7')
  .onChange((value: number) => {
    this.compressionRatio = value;
  })
}
.width('100%')
.height(40)
```

### 2. 百分比显示样式修改
```typescript
// 修改前
Text(`${Math.round(this.compressionRatio * 100)}%`)
  .fontSize(14)
  .fontColor('#182431')
  .fontWeight(FontWeight.Medium)

// 修改后
Text(`${Math.round(this.compressionRatio * 100)}%`)
  .fontColor(Color.Black)
  .opacity(0.6)
```

### 3. 布局结构调整
```typescript
// 修改前 - 简单布局
Column() {
  // ... 滑动条代码
}
.height(100)
.width('100%')

// 修改后 - 与VolumePanelView一致的布局
Column() {
  // 压缩强度滑动条
  Column() {
    Row() {
      Text($r('app.string.compression_strength'))
      Text(`${Math.round(this.compressionRatio * 100)}%`)
        .fontColor(Color.Black)
        .opacity(0.6)
    }
    .justifyContent(FlexAlign.SpaceBetween)
    .width('100%')
    .height(48)

    Stack({ alignContent: Alignment.Top }) {
      // ... 滑动条代码
    }
    .width('100%')
    .height(40)
  }
  .margin({ bottom: 12 })
  .alignItems(HorizontalAlign.Start)
  .width('100%')
  .height(88)
}
.width('100%')
```

## 技术实现细节

### 1. 滑动条容器
- 使用`Stack({ alignContent: Alignment.Top })`包裹滑动条，与音频流音量滑动条保持一致
- 设置固定高度`.height(40)`确保布局一致性

### 2. 滑动条属性
- **style**: `SliderStyle.InSet` - 内凹样式，提供更好的视觉反馈
- **reverse**: `false` - 正常方向（从左到右）
- **trackThickness**: `20` - 轨道厚度20px，提供更好的触摸区域
- **selectedColor**: `'#0A59F7'` - 使用与音频流音量相同的蓝色

### 3. 交互增强
- **focusable(true)**: 允许滑动条获取焦点
- **defaultFocus(true)**: 默认可以获取焦点
- **focusOnTouch(true)**: 触摸时自动获取焦点，提升触摸体验

### 4. 视觉一致性
- **百分比显示**: 使用黑色60%透明度，与音频流音量百分比显示一致
- **布局间距**: 使用与VolumePanelView相同的布局结构
- **高度设置**: 文本行高度48px，滑动条区域高度40px，总计88px
